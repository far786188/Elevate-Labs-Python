# Task 3 - Web Scraper for News Headlines

## Description

This project is a Python web scraper that collects the latest news headlines from Hacker News and saves them into a text file.

## Tools Used

- Python
- Requests
- BeautifulSoup

## Files

- `scraper.py` - Python source code
- `headlines.txt` - Saved news headlines
- `requirements.txt` - Required Python libraries

## How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Run the program:

```bash
python scraper.py
```

## Output

- Fetches the latest news headlines.
- Displays the headlines in the terminal.
- Saves the headlines in `headlines.txt`.
