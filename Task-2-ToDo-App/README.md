# 📝 Python To-Do List Application

A simple console-based To-Do List application developed using Python.

## ✨ Features

- Add new tasks
- View all tasks
- Remove tasks
- Save tasks to a text file (`tasks.txt`)
- Load saved tasks automatically

## 🛠️ Technologies Used

- Python
- Lists
- Functions
- File Handling
- Exception Handling

## 📂 Project Structure

```
Task-2-ToDo-App/
│── todo.py
│── tasks.txt
│── README.md
```

## ▶️ How to Run

```bash
python todo.py
```
